# Eco-Waste Management System 🌱

A full-stack web application that allows users to schedule eco-friendly waste collection from their homes.

## 🚀 Features
- User Authentication (Login/Signup)
- Schedule Waste Pickup
- Dashboard with Request Tracking
- Waste Segregation Guide
- Smart Scheduling System

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: SQLite

## 📌 How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000

## 🌍 Goal
To promote efficient waste management and encourage eco-friendly practices through digital solutions.
